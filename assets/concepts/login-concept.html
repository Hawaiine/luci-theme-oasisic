⚙️ 壁纸系统实现方案

① 壁纸来源：Bing 每日壁纸 API

// 获取 Bing 每日壁纸（无需 API Key）
const bingUrl = 'https://www.bing.com/HPImageArchive.aspx' +
  '?format=js&idx=0&n=1&mkt=zh-CN';

fetch(bingUrl)
  .then(r => r.json())
  .then(d => {
    const url = 'https://www.bing.com' + d.images[0].url;
    // url 含分辨率参数，自动适配屏幕
    document.querySelector('.login-bg').src = url;
    localStorage.setItem('bgUrl', url);
    localStorage.setItem('bgTime', Date.now());
  });

② 自动更换策略：

// 检查是否需更新（默认 6 小时）
const HOURS = 6;
const lastTime = localStorage.getItem('bgTime');
const elapsed = (Date.now() - parseInt(lastTime)) / 1000 / 3600;
if (!lastTime || elapsed > HOURS) {
  fetchNewWallpaper();
} else {
  // 使用缓存的壁纸
  document.querySelector('.login-bg').src =
    localStorage.getItem('bgUrl');
}

③ 多端适配：Bing 壁纸返回的 URL 含分辨率参数（如 _1920x1080.jpg），前端可替换为 _1920x1080、_1366x768、_1080x1920 来自适应不同屏幕。外层容器用 object-fit: cover 确保铺满。

④ 回退方案：壁纸加载失败时，自动降级为内置渐变色背景（深蓝紫渐变）。
